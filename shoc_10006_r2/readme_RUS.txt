
Описание правок:
;=====ДАЛЬНОСТЬ ОТРИСОВКИ ТРАВЫ=======
   увеличивает лимит дальности отрисовки травы до 250 м, добавляет
   консольную команду для ее регулировки r__detail_radius.
   Лимит 250 можно поменять в console_comm.asm (число 0F0h)

;=========КОНСОЛЬНЫЕ КОМАНДЫ==========
   Включает консольные команды:
   r__detail_radius
   r2_sun_shafts
   r__bloodmarks
   r2_soft_water
   r2_soft_particles

;==========ПЛОТНОСТЬ ТРАВЫ============
   Изменяет лимит плотности травы до 0.02

;=============ТЕНЬ ТРАВЫ==============
   Возвращает работоспособность команде r2_sun_details - 
   теперь трава будет отбрасывать тень

;======ОЧИСТКА РЕНДЕРТАРГЕТА RT0======
   Правка принудительно очищает одну из игровых текстур.
   Включать следует при использовании саншафтов.
   Тогда не будет дурацких следов на небе от геометрии.

;=====ШЕЙДЕР САМОСВЕЧЕНИЯ НА ХУДЕ=====
   Правка заставляет шейдеры свечения работать на худе.

;=======ВИРТУАЛЬНЫЙ РАДИУС ЛАМП=======
   Правка меняет лимит изменения main_virtual_size для ламп
   С ней тени от решеток ламп не будут обрезаться

;==============БЛУДМАРКИ==============
   Правка восстанавливает блудмарки (r__bloodmarks on/off)

;===========ДЕФАЙНЫ ШЕЙДЕРОВ==========
   Служебная правка для шейдеров. Включать, если используете шейдерные плюшки огсе.

;=========РЕГИСТРАЦИЯ СЭМПЛЕРОВ=======
   Служебная правка для шейдеров. Включать, если нужна пыль в саншафтах.

;===РАСШИРЕНИЕ КЛАССА CRENDERTARGET===
   Служебная правка для шейдеров. Включать, если нужна пыль в саншафтах. 

;=========БЛЕНДЕР ТЕПЛОВИДЕНИЯ========
   Служебная правка для шейдеров, добавляет новый тип блендера в движок.
   Включать не стоит. 

;==ТЕКСТУРА ШУМА ВЫСОКОГО РАЗРЕШЕНИЯ==
   Служебная правка для шейдеров. Включать, если нужна пыль в саншафтах. 
   Не работает без РАСШИРЕНИЕ КЛАССА CRENDERTARGET.

;=====ПРАВИЛЬНОЕ ПОЛОЖЕНИЕ СОЛНЦА=====
   Правка включает правильные тени от солнца.

;====ДАЛЬНОСТЬ ПЕРЕКЛЮЧЕНИЯ ЛОДОВ====
   Правка расширяет диапазон переключения лодов (r__geometry_lod) до 3,0

;=====ЧЕТКОСТЬ ТЕКСТУР (MIPBIAS)======
   Правка расширяет диапазон настройки четкости текстур (r2_tf_mipbias) до [-3.0;3,0]

;=РЕГИСТРАЦИЯ СЭМПЛЕРОВ ДЛЯ ПАРТИКЛОВ=
   Служебная правка для шейдеров. Включать для мягких партиклов

;===========ДЕТАЛЬНЫЙ БАМП============
   Правка включает возможность использовать детальный бамп из ЧН.
   Без шейдера безполезна. Шейдер можно взять из ЧН, но его надо
   некоторым образом поменять.

;========тень от ГГ на R2============
   Одна из двух врезок для восстановления тени ГГ. Вторая - в xrGame.dll

;==Дополнительные разрешения карты теней====
   Правка добавляет дополнительные разрешения карт теней - 8192x8192 и 16384x16384.
   Используйте, если вам мало 4096х4096.

;====Расширенная регулировка r2_sun_near====
   Увеличивает верхний предел регулировки r2_sun_near до 150. Для больших значений
   придется увеличивать и разрешение карты теней, иначе будут артефакты.
