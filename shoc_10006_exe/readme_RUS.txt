
Описание правок:
;==========ПОЛНОЭКРАННЫЕ ЗАСТАВКИ============
   Правка растягивает интро-заставки при загрузке на весь экран

;========УВЕЛИЧЕНИЕ ВЕРШИННОГО БУФЕРА========
   Правка снижает вероятность вылета (bytes_need<=mSize) && vl_Count

;====ЧТЕНИЕ БАЗ ДАННЫХ ОТ РУССКОЙ ВЕРСИИ=====
   "Переделывает" английский экзешник в русский.
   Правка заставляет экзешник от английской версии
   без защиты читать архивы от русской версии.

;====РЕГИСТРАЦИЯ ПЕРЕМЕННЫХ ДЛЯ ШЕЙДЕРОВ=====
   Правка позволяет добавлять свои глобальные переменные для шейдеров.
   Служебная правка для шейдеров. Включать, если используете шейдерные плюшки от огсе.

;=======РЕГИСТРАЦИЯ КОНСОЛЬНЫХ КОМАНД========
   Включение консольных команд. В текущем виде добавляет команду
   r2_script_sun_shafts - интенсивность саншафтов, для управления из скриптов   

;============ПОГОДНЫЕ ПАРАМЕТРЫ==============
   Правка добавляет в конфиг погоды параметры:
   sun_shafts - интенсивность саншафтов
   sun_shafts_density - плотность саншафтов
   rain_max_drop_angle - максимальный угол падения капель дождя (по умолчанию 10 градусов)

   При включении правки эти параметры надо будет добавить во все погодные секции.

;======ФИКС ВЕРТИКАЛЬНОЙ СИНХРОНИЗАЦИИ=======
   Восстанавливает работоспособность команды rs_v_sync
